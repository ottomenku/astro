### 4) Jelölgethető ellenőrzőlista (hogy mi van kész / mi hiányzik)

- [ ] Tabok: csak az aktív legyen kék
- [ ] Nap: sárga alapon fehér
- [ ] 1. belső gyűrű (ahová a fényszögek mutatnak): -20% vastagság
- [ ] Jegyeknek megfelelő fok- és dekádbeosztás (külső kör jellegű)
- [ ] Vonal jelölje a bolygók helyzetét
- [ ] Bolygó színek: Nap sárga, Merkúr zöld, Vénusz kék, Mars piros, Jupiter bordó, Szaturnusz lila, többi fekete
- [ ] 2. belső gyűrű (bolygók/ház): +20% vastagság
- [ ] Percek + órák léptetése egy sorban
- [ ] Napok léptetése mellett hónapok léptetése
- [ ] Legkülső kör ne kelljen; a zodiákus legyen a legkülső
- [ ] Zodiákus gyűrűben dekád színezés:
  - [ ] 1. dekád: fekete (negatív) / fehér (pozitív)
  - [ ] 2. dekád: lila (fix) / sárga (kardinális) / világoskék (változó)
  - [ ] 3. dekád: kék (vizes) / piros (tüzes) / barna (földies) / zöld (levegős)
